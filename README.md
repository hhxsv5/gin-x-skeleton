# The skeleton for [Gin-X](https://github.com/hhxsv5/gin-x) framework.

# Usage

1. Copy this project into your project, except `.gitignore`.

2. Copy config.toml.example into config.toml.

3. Start Http Server.
```bash
# http://127.0.0.1:8000 && http://127.0.0.1:5200
./build-debug.sh

# OR

# http://127.0.0.1:5200
go run main.go
```
